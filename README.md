# Mindful - Mental Wellness Website

A clean and modern mental wellness website providing resources for mental health, stress management, self-care, and support.

## 📋 Project Overview

Mindful is a static HTML/CSS website designed to help users learn about and manage their mental wellness through various resources and information.

## 🎯 Features

- **Home Page** - Welcome and introduction to the platform
- **Health Section** - Mental health information and resources
- **Self-Care** - Self-care tips and wellness practices
- **Stress Management** - Techniques and strategies for managing stress
- **Support Resources** - Links and information for mental health support

## 🚀 Getting Started

### Prerequisites
- A modern web browser
- A text editor or IDE for editing

### Installation

1. Clone the repository:
```bash
git clone https://github.com/yourusername/mindful-web-project.git
cd mindful-web-project
```

2. Open `index.html` in your web browser or use a local server:
```bash
# Using Python 3
python -m http.server 8000

# Using Node.js (with http-server)
npx http-server
```

3. Navigate to `http://localhost:8000` in your browser

## 💻 Development

### Technology Stack
- **HTML5** - Semantic markup

Edit the HTML files in the root and `pages/` directory to change content.
Modify CSS files in the `css/` directory to update styling.

## 🌐 Deployment

### GitHub Pages
1. Push your code to GitHub
2. Go to Settings → Pages
3. Select main branch as source
4. Your site will be published at `https://yourusername.github.io/mindful-web-project`
